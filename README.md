# 25 Proyectos con CNN Explainer

Página web educativa que documenta 25 proyectos prácticos para explorar el funcionamiento de las Redes Neuronales Convolucionales (CNN) usando la herramienta interactiva **CNN Explainer** del Polo Club de Georgia Tech.

https://duvanobando.github.io/cnn-explainer-proyectos/

## 🔗 Herramienta de referencia

[CNN Explainer - poloclub.github.io/cnn-explainer](https://poloclub.github.io/cnn-explainer/)

## 📚 Estructura del proyecto

- `index.html` — Página principal con el índice de los 25 módulos
- `modulos/` — Una página HTML por cada proyecto (módulo 01 al 25)
- `css/` — Hojas de estilo
- `js/` — Scripts de JavaScript
- `img/` — Capturas de pantalla y recursos visuales organizados por módulo

## 🎯 Objetivo

Comprender, a través de la exploración visual sin necesidad de código, cómo las CNN procesan imágenes capa por capa: convoluciones, ReLU, MaxPooling, flatten y capas densas.
